## Examples
> "I'm just kicking off a one-off coding task."

> 我只是啟動一個**一次性的** coding 任務。

> "This is a one-off — we don't usually do exceptions."

> 這是**一次性的**例外——我們通常不破例。

> "Let's not make this a one-off; let's build it into the process."

> 別把這做成**一次性**——把它放進流程裡。

## Usage tip
**對比 recurring（循環的）/ regular（規律的）**——強調「這件事只做一次」。英式英文常見，現在已全球化。

兩種詞性：
- **形容詞**：`a one-off task / event / payment / exception`
- **名詞**：`a one-off`（一次性事件本身）

常見搭配：
- `one-off task` 一次性任務（最常見）
- `one-off payment / fee` 一次性付費
- `one-off event` 一次性活動
- `not a one-off`（強調不要當一次性事件）

對比：
- `recurring` 循環的（每月／每週）
- `regular` 規律的
- `repetitive` 重複的（負面含義）
- `ad hoc` 臨時的（拉丁文）
