## Examples

> "Product engineers do all the things software engineers do, but they talk directly to users, work autonomously, and are opinionated about what to work on."

> Product engineer 做的事跟一般軟體工程師一樣，但他們會直接跟 user 對話、自主工作，並且對該做什麼很有意見。

> "Our product framework is opinionated — it doesn't try to be flexible for every use case."

> 我們的產品框架很「有定見」——不會試著去配合每個 use case。

## Usage tip

**⚠️ Connotation trap**：中文「有強烈意見的」聽起來帶負面（固執、不好溝通），但英文 **opinionated 在科技／商業情境多半是正面**——指「有判斷力、不是 wishy-washy」。

- 正面用法（PostHog 這篇）：`opinionated about what to work on` = 對優先順序有清楚主張
- 形容軟體/框架：`an opinionated framework` = 預設立場明確，不給太多選擇（Rails 是 opinionated；Django 沒那麼）
- 負面用法：`stubbornly opinionated` 或 `too opinionated to listen` 才偏負面

**常用搭配**：`opinionated about X` / `opinionated take` / `opinionated framework`

## Synonyms / Related

decisive, assertive, principled（同正面） · stubborn, dogmatic（同負面，要避開）
