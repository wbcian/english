## Usage tip
**`-pilled` 後綴**是 2020+ 才在網路上爆紅的構詞，源自《駭客任務》紅藍藥丸的 meme：

| 例子 | 意思 |
|---|---|
| AGI-pilled | 信 AGI 願景 |
| black-pilled | 對未來絕望（負面） |
| based-pilled | 認同反主流觀點 |
| coffee-pilled | 咖啡因依賴症 |
| Claude-pilled | 沉迷 Claude（自嘲常用） |

**寫法**：有沒有連字號都可以（`AGI pilled` / `AGI-pilled`），連字號比較常見、比較好讀。

⚠️ **超不正式**——formal email、論文、面試**完全不能用**。Twitter、Slack 閒聊、播客全部 OK。
