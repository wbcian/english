## Examples
> "This tutorial is considered the canonical guide to REST APIs — everyone links to it."

> 這篇教學被公認是 REST API 的標準指南——大家都會引用它。

> "Always set a canonical URL so search engines know which version of the page is the 'real' one."

> 一定要設定 canonical URL，這樣搜尋引擎才知道哪個版本才是「正版」頁面。

## Usage tip
字源：`canon`（教規、正典，尤指聖經正典或某領域公認的經典作品清單）→ `canonical`＝屬於正典的，引申為「公認標準、大家都認同的範本」。
學術／日常：`the canonical example` / `the canonical text`——最具代表性、大家都拿來當範例的那個，語氣比 `standard` 更帶「權威、公認」的味道。
技術／工程超高頻：`canonical URL`（SEO 用語，告訴搜尋引擎哪個網址是正版，避免重複內容互相稀釋排名）、`canonical form / canonical data model`（規範化後的唯一標準格式）。
反義／對比：`non-canonical`、`alternative`。

## Synonyms
