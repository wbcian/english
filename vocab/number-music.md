## Examples

> "You're gonna pick a slow number, all right?"

> 你給我選一首慢歌，懂嗎？

> "Let's do one more number before we wrap up."

> 收場前再來一首吧。

> "That opening number brought the house down."

> 開場那首炸翻全場。

> "We are not doing a Beyoncé song for the opening number."
> 我們開場曲不會選 Beyoncé 的歌。

## Usage tip

- **核心限制**：這個「歌曲／曲目」的用法**只在 theater / musical / jazz / live performance 圈**有效。日常聊「歌」還是用 song。
- **常見搭配**：
  - opening number — 開場曲
  - closing number — 結尾曲
  - dance number — 舞蹈段落
  - musical number — 音樂劇橋段
  - slow number — 慢歌
  - showstopping number — 全場最炸的那首
- 衍生：do a number on sb（成語）= 把某人整慘 / 騙慘（語意完全不同，別混）。

## Cultural Notes

來自 vaudeville 與 Broadway 傳統——一場表演有好幾個「numbers」（編號的節目段落），所以叫 number。

## Synonyms / Related

song, piece, tune, track（錄音用）, set（DJ／樂團一連串）
