## Examples
> "Making sure our marketing, sales, and finance teams are bought in and that we're all rowing the same direction."

> 確保 marketing、sales、finance 團隊都買單、**大家一起朝同方向划**。

> "If half the team is rowing in the opposite direction, the boat doesn't move."

> 如果一半人**往反方向划**，船根本不會動。

## Usage tip
**划船隊比喻**——畫面感超強：賽艇上 8 個人必須節奏一致才會前進，一個人亂划全隊翻車 🚣。

常見變體：
- `rowing the same direction`（Cat Wu 版本）
- `rowing in the same direction`（更常見）
- `pulling in the same direction`（同義變體）
- `paddling in the same direction`（划獨木舟版）

對比 `alignment` 這個詞——`rowing in the same direction` **更有畫面、更口語**。寫公司 doc 時兩個都用，效果更立體。

反義表達：
- `pulling in different directions` 各拉各的
- `working against each other` 互相扯後腿

### Cian 套用場景
- "Before this offsite, the team wasn't really **rowing in the same direction**."
- "Once we set the OKRs, we were all **pulling in the same direction**."

## Synonyms
aligned, on the same page, in sync, pulling together, marching to the same beat
