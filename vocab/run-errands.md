## Examples
> "I need to run a few errands before the stores close."

> 商店關門前，我得跑幾個地方辦事。

## Usage tip
`errand` 是短程、具體的小任務，例如買東西、領乾洗或寄包裹。最常用複數：`run errands`、`run a few errands`。替別人跑腿可說 `run an errand for someone`。
