## Examples
> "I already run my own routers and access points at home."

> 我家裡本來就自己在跑路由器和無線基地台。

> "The Wi-Fi drops in the back room — we probably need another access point."

> 後面那間房 Wi-Fi 會斷，我們大概得再加一台無線基地台。

## Usage tip
⚠️ **重音陷阱**：名詞 **AC**-cess /ˈækses/ 重音在**第一**音節；動詞 to ac**CESS** /əkˈses/ 才在第二音節。`access point` 是名詞，所以一律唸 **AC**-cess point。
連音：`an access point` 唸成 a-**nak**-cess point（`an` 的 /n/ 滑進 access）。
口語一律縮寫成 **AP**（唸字母 /eɪ ˈpiː/），複數 APs。書面第一次寫全名再標縮寫：`access point (AP)`。
Wi-Fi 硬體三兄弟分工要分清楚：`router`（路由器，決定封包往哪走、通常也接外網）／`switch`（交換器，有線埠擴充）／`access point`（把有線網路變成 Wi-Fi 訊號，**本身不做路由**）。家用機常三合一，企業環境才拆開——這正是企業級網通品牌的主場。
廣義非技術用法也存在：`an access point for new users`（新使用者的入口／切入點），但日常英文更常說 `entry point` / `way in`。

## Synonyms
AP（縮寫）, wireless access point, WAP；相關 router, switch, gateway, mesh node
