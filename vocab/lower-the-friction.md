## Examples
> "This really tight process lowers the friction for any engineer to ship something."

> 這個很緊的流程**降低任何工程師出貨的摩擦力**。

> "One-click login lowered the friction so much that signups doubled."

> 一鍵登入大幅**降低了摩擦**，註冊數翻倍。

> "Adding a confirmation dialog just adds friction without value."

> 加確認對話框只是**增加摩擦**，沒帶來價值。

## Usage tip
**矽谷 UX / 流程設計詞**——把「做某件事的難度」物理化成「摩擦力」。

兩個對稱用法：
- `lower / reduce the friction` 降低摩擦
- `add friction` / `introduce friction` 增加摩擦（負面）

完整句式：
- `[X] lowers the friction for [N] to [V]`
- `[X] adds friction without [N]`（批評設計）

中文沒有完全對應詞——「降低門檻」最接近但少了「物理直觀感」。

### Cian 工作場合套用
- This script **lowers the friction for new engineers to set up locally**.
- Our CI process **adds way too much friction for shipping hotfixes**.
