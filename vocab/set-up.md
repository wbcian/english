## Examples
> "Did you ever think it would help set up for one of the biggest viral moments of your career?"
> 你曾想過這會為你職涯上最爆紅的瞬間鋪路嗎？

> "Taking on leadership roles in college set me up to advance quickly in my career."
> 大學擔任領導角色為我之後快速升遷鋪了路。

## Usage tip
**set up** 多義詞，看語境：

| 意思 | 例子 |
|---|---|
| 為…鋪路 / 創造條件 | "X set me up for success." |
| 安排（會議） | "Let's set up a meeting." |
| 開設、建立 | "She set up her own business." |
| 開店、開張 | "set up shop" |

## set someone up — 五種主要型式

| 模式 | 意思 | 例子 |
|---|---|---|
| set sb up **for** + 結果 | 鋪路（中性／負面） | "This internship will set you up for a great career." |
| set sb up **with** + 人 | 撮合對象 | "My friend set me up with her cousin." |
| set sb up **with** + 物 | 配備 / 提供 | "The hotel set us up with a free breakfast." |
| set sb up（被動 + 警戒） | 陷害 | "I didn't do it — I was set up!" |
| set sb up **in** + 場所/事業 | 出資建立 | "His parents set him up in his own apartment." |

### Gotcha
- 語氣決定一切：同一句 "set me up" 可能是撮合也可能是陷害
- 主動/被動切換意義：
  - "I set him up." → 多半是撮合
  - "I was set up." → 多半是被陷害
- 名詞 **a setup** 同樣多義：圈套 / 相親 / 設備配置

## Synonyms
prepare, pave the way, lay the groundwork (for), arrange, fix up (sb with sb), frame (= 陷害)
