## Examples

> "Do some legwork. Once a call is set up, I write a one pager that summarizes where we are at."

> 做點事前功課。通話安排好後，我會寫一份 one pager 總結目前狀態。

> "I did all the legwork for this report — interviews, data pulls, the works."

> 這份報告所有的前置功夫都是我做的——訪談、撈數據、什麼都做了。

## Usage tip

- **不可數**，所以不會說 `a legwork` 或 `legworks`，要說 `the legwork` / `some legwork` / `a lot of legwork`
- 常見搭配：`do the legwork` / `put in the legwork`
- TOEIC Part 5/6/7 商業文偶爾出現，多半描述「前置準備工作」
- 字面是「腿的工作」（跑來跑去做基礎調查），延伸成所有耗時的準備功夫

## Synonyms / Related

groundwork, prep work, footwork, due diligence
