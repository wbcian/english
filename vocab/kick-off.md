## Examples
> "I tend to use Cloud Code when I'm just kicking off a one-off coding task."

> 我會用 Cloud Code，當我**啟動**一次性的 coding 任務時。

> "Mobile lets you kick off these tasks on the go."

> Mobile 讓你能在外面**啟動**這些任務。

> "Let's kick off the meeting — we have 30 minutes."

> 我們**開始**會議吧——只有 30 分鐘。

> "The project kicks off next Monday."

> 這個專案下週一**啟動**。

## Usage tip
**萬用工作動詞**。出處：足球／橄欖球「開球」⚽，引申為「啟動任何東西」。

兩種詞性：
- **動詞片語**：`kick off [task / meeting / project]`
- **名詞**：`kick-off`（連字號）or `kickoff`（一字）→ "the kickoff is at 3pm"

常見搭配：
- `kick off a task / project / meeting` 啟動 ~
- `kick off with N` 用 ~ 開場 → "Let's kick off with a quick intro."
- `kick things off` 開啟整件事 → "Cat, do you want to kick things off?"

**替換練習**：以前你會說 `start a task` → 改成 `kick off a task`，職場英文升一級。
