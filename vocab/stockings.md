## Usage tip — 中文「絲襪」整個 family

| 英文 | 款式 | 中文 |
|---|---|---|
| **stockings** /ˈstɒkɪŋz/ | 大腿長度、單腳一隻 | 長筒絲襪 |
| **pantyhose** /ˈpæntihoʊz/ | 連身、含內褲（美式） | 連褲襪 |
| **tights** /taɪts/ | 連身、含內褲（英式 / 厚款） | 褲襪 / 內搭褲 |
| **fishnets** /ˈfɪʃnɛts/ | 漁網狀，性感／戲劇感 | 漁網襪 |
| **thigh-highs** | 到大腿、自帶矽膠帶 | 大腿襪 |
| **knee-highs** | 到膝蓋下 | 及膝襪 |
| **nylons** /ˈnaɪlɒnz/ | 口語稱透膚絲襪 | 尼龍絲襪 (informal) |
| **hosiery** /ˈhoʊʒəri/ | 「襪類」總稱 | 襪品（百貨櫃位用詞）|

### 怎麼選
| 你想說 | 用哪個 |
|---|---|
| 辦公室透膚絲襪 | pantyhose（美）/ tights（英）|
| 復古、到大腿 | stockings |
| 漁網款 | fishnets |
| 黑色不透光內搭 | opaque tights |

### ⚠️ 三個地雷
1. **stocking ≠ sock**：sock 是一般襪子；stocking 才是絲襪。
2. **pantyhose 在英國很少用**，英國人多半叫 tights。
3. **Christmas stocking** = 聖誕襪（壁爐那個），同字不同物。
