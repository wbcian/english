## What it means

`X-shy` = 「避開 X」、「怕 X」、「遇 X 就退縮」。是一個**極度好用、可以隨手造詞**的 compound adjective 字尾。

## Examples（lesson 給的全家族）

> "war-shy fucker" — 避戰的傢伙；引申為膽小鬼。
> "She's always been camera-shy." — 她一直都怕鏡頭。
> "He's a bit work-shy." — 他有點偷懶（逃避辛苦工作）。
> "I'm commitment-shy after my last relationship." — 上一段感情之後我有點怕承諾。

## Productive usage（自由造詞）

| 字 | 意思 |
|---|---|
| war-shy | 避戰；本質膽小 |
| camera-shy | 怕被拍 |
| work-shy | 偷懶；好逸惡勞 |
| commitment-shy | 怕進入認真關係 |
| gun-shy | 一朝被蛇咬（原指槍響後就怕了） |
| publicity-shy | 不愛曝光 |
| spotlight-shy | 怕站上舞台中心 |
| media-shy | 不愛接受媒體採訪 |
| risk-shy | 不愛冒險（vs risk-averse 比較正式） |

> 規則：選一個你想避開的事物 X，加 `-shy`，就成立。

## Usage tip

- **連字號**：`X-shy` 一律加 hyphen（compound adjective）。
- **詞性**：永遠是 adjective。可作 attributive（war-shy soldier）或 predicative（he's camera-shy）。
- **語氣**：偏 informal，會帶一點批評／嘲弄味（war-shy, work-shy 都不是讚美）。

## Synonyms / Related

`-averse`：risk-averse, conflict-averse — 比較中性、formal。
`-phobic`：claustrophobic — 醫學／極度害怕等級。
`-shy` 在三者中最口語、最隨意。
